**Recipes Generator**

A simple and interactive recipes generator web app built with HTML, CSS, and JavaScript.  
It uses the TheMealDB API to fetch and display delicious recipes from around the world.

---

**Features**

-  Search for meals by name or keyword
- View full recipe details and its ingredients.
- Watch recipe videos (YouTube link) for all recipes.
- Background cooking music and Fullscreen mode.

---

**How it works**

- Type a meal name in the search bar
- Results are displayed as cards
- Click any meal and scroll down to see full details
- Enjoy your excellent cooking.

---

**languages used**

- HTML
- CSS
- JavaScript
